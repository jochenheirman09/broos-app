# **App Name**: RoleVerify

## Core Features:

- User Registration: Allow users to register with an email, password, and role selection (player, staff, responsible).
- Role-Based Access Control: Enforce user roles (player, staff, responsible) to control access to features.
- Email Verification: Require users to verify their email address upon registration.
- Personalized Dashboard: Display a welcome message with the user's name and role.
- Club Creation Check: Check if a 'responsible' user has a linked Club; prompt for Club creation if needed.
- Club Creation Form: Provide a form for 'responsible' users to create a new Club, saving the club data to the database.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy feel.
- Background color: Very light blue (#E8EAF6) for a clean interface.
- Accent color: Amber (#FFC107) to highlight important actions and elements.
- Font: 'Inter', a sans-serif typeface, should be used for all text.
- Use simple and clear icons to represent roles and actions.
- Design a clean and intuitive layout for forms and data display, suitable for native iOS/Android mobile app.
- Use subtle transitions for a smooth user experience, optimized for native mobile performance.
